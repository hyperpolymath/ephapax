* Works in standalone test files
* Fails only inside complex inductive proofs
* No newer Rocq version available via opam
* 15+ tactic combinations attempted, all fail
* Blocks: `subst_preserves_typing` and `preservation` proofs

Resolution: wait for Rocq 9.2, or restructure proofs to avoid `nth_error` on
`list (ty * bool)` (use separate type/flag lists instead).
